Course Introduction — 강좌 소개

이 강좌은 DGIST EECS의 Sunghoon Im 교수님이 개설한 CSE404: Introduction to Computer Vision 과목의 첫 번째 슬라이드입니다. 컴퓨터 비전이란 무엇인지, 왜 배워야 하는지, 어떤 내용을 다루는지를 소개합니다.
상세 설명

컴퓨터 비전(Computer Vision)은 컴퓨터가 이미지나 영상을 "본다"고 할 수 있도록 만드는 학문 분야입니다. 우리가 눈으로 세상을 보고 "저건 고양이야", "저 건물은 3층이야" 같은 판단을 하듯, 컴퓨터도 픽셀 데이터로부터 의미있는 정보를 추출할 수 있도록 하는 것이 목표입니다.


이 과목은 컴퓨터 비전의 이론적 기초부터 실용적 응용까지를 폭넓게 다룹니다. 이미지 형성 원리, 특징 추출, 객체 인식, 3D 재구성, 딥러닝 기반 방법 등을 배우게 됩니다.

핵심 용어
영어한국어설명
Computer Vision 컴퓨터 비전 컴퓨터가 이미지/영상을 이해하도록 하는 학문 분야
CSE404 과목 코드 DGIST 전자전기컴퓨터공학부 개설 과목
컴퓨터 비전은 인공지능(AI)의 핵심 하위 분야 중 하나입니다. 자율주행, 의료 영상, 메타버스, 로봇 등 최첨단 기술의 근간이 바로 컴퓨터 비전입니다.

강좌 목표 (Course Objectives)

이 과목의 목표는 컴퓨터 비전의 이론적(theoretical) 및 실용적(practical) 기초를 확립하고, 다양한 컴퓨터 비전 응용에 대한 직관(intuition)을 기르는 것입니다.
상세 설명

강좌은 크게 세 가지 축으로 구성됩니다:


Fundamentals
기초 원리
+
Approaches
접근 방법
+
Applications
응용 분야

1. Fundamentals (기초 원리): 빛이 어떻게 이미지로 변환되는지, 카메라의 기하학적/물리적 모델링 등 기본적인 원리를 배움니다.


2. Approaches (접근 방법): 픽셀 데이터로부터 의미를 추출하는 구체적인 기법들(특징 추출, 매칭, 인식, 재구성 등)을 학습합니다.


3. Applications (응용 분야): 배운 기법들이 실제 어디에 활용되는지(로봇, 자율주행, 의료 등)를 살펅봉니다.

"직관(intuition)"이란 단순히 공식을 외우는 것이 아니라, 문제를 보았을 때 "아, 이건 이런 방법으로 풀면 되겠다"고 자연스럽게 떠올리는 능력을 말합니다. 이 강좌이 그런 직관을 키우는 것을 목표로 합니다.

이미징 (Imaging): 빛을 픽셀로 변환하기

컴퓨터 비전을 이해하려면, 먼저 이미지가 어떻게 만들어지는지를 알아야 합니다. 3D 세계의 빛이 카메라를 통해 2D 픽셀 행렬(이미지)로 변환되는 과정이 바로 이미징(Imaging)입니다.
상세 설명

우리가 사진을 찍는 과정을 생각해 보세요:


1. 3D 셀(Scene): 실제 세계에는 집, 나무, 사람 등이 3차원으로 존재합니다.


2. 빛(Light): 태양이나 조명 등에서 나온 빛이 물체 표면에서 반사됩니다.


3. 카메라: 반사된 빛이 카메라 렌즈를 통과하여 센서 평면에 도달합니다.


4. 픽셀 행렬(Pixel Matrix): 센서에 도달한 빛의 양이 숫자로 변환되어 행렬 형태로 저장됩니다. 예를 들어:

217  191  252  255  239
102   80  200  146  138
159   94   91  121  138
179  106  136   85   41
115  129   83  112   67
 94  114  105  111   89

이런 숫자 행렬이 바로 우리가 보는 "이미지"입니다. 각 숫자는 해당 위치의 밝기(또는 색상) 값을 나타냅니다.

핵심 용어
영어한국어설명
Imaging 이미징 빛을 픽셀 행렬(이미지)로 변환하는 과정
Pixel Matrix 픽셀 행렬 이미지를 구성하는 숫자 배열. 각 원소가 하나의 픽셀 밝기를 나타낼
3D Scene 3차원 장면 실제 세계의 3차원 공간에 존재하는 물체들의 배치
비유: 이미징은 마치 우리가 3D 세계를 창문을 통해 바라보면서 그 장면을 스케치북에 그리는 것과 같습니다. 3D 정보가 2D 평면으로 압축되는 것이죄.

Computer Vision: 빛을 의미로 변환하기

이미징이 "빛 → 픽셀"의 과정이라면, 컴퓨터 비전은 그 반대 방향입니다. 즉 "픽셀 → 의미"를 추출하는 것이 컴퓨터 비전의 핵심입니다.
상세 설명

컴퓨터 비전은 픽셀 행렬(이미지)로부터 다양한 의미(meaning)를 추출합니다. 여기서 "의미"란 기하학적(geometric) 정보와 의미론적(semantic) 정보를 모두 포함합니다:


  • Objects (객체): 이미지 안에 어떤 물체가 있는지 인식
  • Shape/Geometry (형상/기하): 물체의 3D 형태 파악
  • Semantics (의미론): 장면의 의미 이해 (예: "이것은 교실이다")
  • Material (재질): 물체의 표면 속성 (금속? 나무? 유리?)
  • Motion (운동): 물체가 어떻게 움직이는지
  • 3D Pose (3D 자세): 물체의 3차원 위치와 방향
핵심 용어
영어한국어설명
Semantic 의미론적 "무엇인지"에 대한 이해. 예: 이것은 고양이, 저것은 자동차
Geometric 기하학적 "어디에 어떤 모양으로" 있는지에 대한 정보
3D Pose 3D 자세 객체의 3차원 공간에서의 위치(x,y,z)와 회전(roll, pitch, yaw)
컴퓨터 비전의 핵심을 한 문장으로 요약하면: "빛(light)을 의미(meaning)로 변환하는 것"입니다. 이것만 기억하세요!

비정칠 역문제 (Ill-posed Inverse Problem)

컴퓨터 비전은 본질적으로 비정칠 역문제(ill-posed inverse problem)입니다. 하나의 2D 이미지를 만들어낼 수 있는 3D 장면이 무수히 많기 때문입니다.
상세 설명

역문제(Inverse Problem)란 결과로부터 원인을 추론하는 문제입니다. 컴퓨터 비전에서는 2D 이미지(결과)로부터 3D 세계(원인)를 알아내야 합니다.


"비정칠(ill-posed)"이란 유일한 해(solution)가 존재하지 않는다는 뜻입니다:

  • 동일한 2D 이미지를 만들어낼 수 있는 서로 다른 3D 장면이 무수히 존재
  • 따라서 추가적인 제약 조건(constraints)이나 세계에 대한 사전 지식(prior knowledge)이 필요

비유를 들어보겠습니다. 그림자 없이 누군가 "동그란 그림을 그려줄래"라고 하면, 여러분은 많은 가능한 그림을 상상할 수 있습니다. 하지만 "밝은 곳에 놀고 있는 고양이"라고 추가 정보를 주면 가능성이 줄어듭니다. 컴퓨터 비전에서도 마찬가지로, 추가 제약이 핵심입니다.

핵심 용어
영어한국어설명
Ill-posed Problem 비정칠 문제 해가 유일하지 않거나 존재하지 않거나 입력에 연속적으로 의존하지 않는 문제
Inverse Problem 역문제 관측된 결과로부터 원래 상태를 추론하는 문제
Constraints / Prior 제약 조건 / 사전 지식 문제를 풀기 위해 추가로 필요한 조건이나 정보
이것이 컴퓨터 비전이 어려운 근본적인 이유입니다. 3D에서 2D로의 변환(이미징)은 정보를 잃어버리는 과정이고, 그 잃어버린 정보를 복원하는 것은 원래 불가능하기 때문입니다.

Computer Vision vs. Computer Graphics

컴퓨터 비전과 컴퓨터 그래픽스는 정반대의 방향을 다룹니다. 그래픽스는 심벌(3D 모델) → 이미지이고, 비전은 이미지 → 심벌(의미)입니다.
상세 설명

두 분야의 관계를 정리하면:


Computer Graphics (컴퓨터 그래픽스)

  • 방향: 3D 셀 → 2D 이미지
  • 심벌(Symbol)로부터 이미지를 생성합니다
  • 예: 게임 렌더링, 영화 VFX, 3D 모델링
  • 입력: (cube, size, x₀, y₀, z₀, θₓₚ, θₓ₨, θₚ₨, ...) 같은 기하학적 파라미터

Computer Vision (컴퓨터 비전)

  • 방향: 2D 이미지 → 의미/3D 정보
  • 픽셀 행렬로부터 객체, 형상, 운동 등을 파악합니다
  • 예: 얼굴 인식, 자율주행차 장애물 감지

쉽게 말해, 그래픽스는 "만드는 것"이고 비전은 "이해하는 것"입니다. 두 분야는 서로 보완적인 관계에 있으며, 최근에는 NeRF, 3D Gaussian Splatting 등 두 분야의 경계가 흐려지는 연구가 많습니다.

핵심 용어
영어한국어설명
Computer Graphics 컴퓨터 그래픽스 3D 모델/심벌으로부터 2D 이미지를 생성하는 분야
Symbol 심벌 객체의 기하학적 파라미터(위치, 크기, 회전 등)로 표현한 것
Rendering 렌더링 3D 모델을 2D 이미지로 변환하는 과정 (그래픽스의 핵심 과정)
기억법: 그래픽스는 "심벌 → 이미지" (창조), 비전은 "이미지 → 심벌" (분석). 둘 다 빛과 기하학을 다루지만 방향이 반대입니다.

기초: 이미지 형성 (Image Formation)

과목의 첫 번째 큰 주제는 이미지 형성(Image Formation)입니다. 빛이 어떻게 픽셀 행렬(이미지)로 변환되는지를 4가지 하위 주제로 나누어 배웁니다.
상세 설명

이미지 형성은 다음 4가지 하위 주제로 구성됩니다:


(0) Primitives and Transformations (기본 요소와 변환)

점, 선, 평면 등 기본 기하학적 요소와, 이들을 변환(회전, 이동, 크기 변환 등)하는 방법을 배웁니다. 모든 컴퓨터 비전의 수학적 기반이 됩니다.


(1) Photometric Image Formation (광도 기반 이미지 형성)

빛원(light source)에서 나온 빛이 물체 표면(surface)에 부딛히면 어떻게 반사되는지를 다룹니다. 표면의 법선 방향(n), 재질 특성 등이 반사 양상에 영향을 미칩니다.


(2) Geometric Image Formation (기하학적 이미지 형성)

렌즈(optics)를 통해 3D 공간의 점이 2D 센서 평면의 어디에 맺히는지를 다룹니다. 핀홀 카메라 모델, 투영 변환 등이 핵심입니다.


(3) Image Sensing Pipeline (이미지 센싱 파이프라인)

센서에 도달한 빛이 전기 신호로 변환되고, 디지털 데이터로 변환되는 과정입니다. CCD/CMOS 센서, 베이어 패턴, ISP 등을 다룹니다.

핵심 용어
영어한국어설명
Photometric 광도계측적 빛의 밝기와 색상에 관한 것
Geometric 기하학적 공간적 위치와 형상에 관한 것
Optics 광학 / 렌즈 빛의 경로를 제어하는 렌즈 시스템
Sensor Plane 센서 평면 카메라 내부에서 빛을 받아들이는 평면 (CCD/CMOS)
Surface Normal 표면 법선 물체 표면에 수직인 방향 벡터 (̂n)
이미지 형성을 이해하는 것은 컴퓨터 비전의 가장 기본적인 토대입니다. "이미지가 어떻게 만들어지는지"를 모르면, "이미지로부터 무엇을 알아낼 수 있는지"도 알 수 없습니다.

과제/접근법 개요 (Tasks & Approaches)

기초를 배운 후에는 실제로 픽셀 행렬(이미지)를 의미로 변환하는 구체적인 과제(Tasks)와 접근법(Approaches)을 배웁니다. 크게 4가지 영역으로 나닅니다.
상세 설명

(1) Computational Imaging (계산 이미징)

이미지의 품질을 개선하거나 새로운 이미지를 생성하는 기술입니다. 예를 들어 흐린 사진을 선명하게 만들거나, 저해상도 이미지를 고해상도로 변환하는 것입니다.


(2) Image Feature (이미지 특징)

이미지에서 의미있는 정보를 추출하는 기술입니다. 예를 들어 두 장의 사진에서 같은 건물을 찾아내려면, 각 이미지에서 "특징적인 점들"을 찾아서 매칭해야 합니다.


(3) Recognition (인식)

이미지 속 객체가 무엇인지 파악하는 기술입니다. Semantic Segmentation(의미론적 분할), Classification(분류), Object Detection(객체 검출), Instance Segmentation(인스턴스 분할) 등이 포함됩니다.


(4) 3D Reconstruction (3D 재구성)

2D 이미지들로부터 3D 구조를 복원하는 기술입니다. 여러 장의 사진을 이용해 건물의 3D 모델을 만들는 것이 대표적입니다.

이 4가지 영역은 서로 독립적이지 않습니다. 예를 들어, 3D 재구성을 하려면 이미지 특징 매칭이 필요하고, 인식을 하려면 계산 이미징으로 전처리한 깨끗한 이미지가 도움이 됩니다.
Page 09

과제별 세부 토픽

앞 페이지에서 소개한 4가지 영역에 속하는 구체적인 세부 주제들을 정리합니다. 이 과목에서 학기에 걸쳐 다룰 내용의 미리보기입니다.
(1) Computational Imaging (계산 이미징)
  • Super Resolution (SR): 저해상도 이미지를 고해상도로 변환
  • Deblurring: 흔들린/흐린 이미지를 선명하게 복원
  • Denoising: 이미지에서 닅즈(잡음)를 제거
  • HDR: 높은 다이나믹 레인지 이미지 생성
(2) Image Feature (이미지 특징)
  • Edge (에지): 이미지에서 밝기가 급격히 변하는 경계선
  • Corner (코너): 두 개 이상의 에지가 만나는 점
  • Key-points & Descriptors: 특징적인 점과 그 주변 정보를 기술한 벡터
  • HOG (Histogram of Oriented Gradients): 그라디언트 방향의 히스토그램을 이용한 특징 기술자
(3) Recognition (인식)
  • Classification (분류): 이미지가 어떤 카테고리에 속하는지 판단
  • Detection (검출): 이미지 속 객체의 위치와 종류를 찾아낼
  • Segmentation (분할): 픽셀 단위로 객체의 영역을 구분
  • Tracking (추적): 영상에서 객체의 움직임을 시간에 따라 추적
(4) 3D Reconstruction (3D 재구성)
  • Stereo Matching: 좌우 두 장의 이미지로부터 깉이(depth)를 추정
  • Multi-view Stereo: 여러 각도의 이미지로부터 3D 모델 생성
  • Structure-from-Motion (SfM): 움직이는 카메라의 영상으로부터 3D 구조와 카메라 위치를 동시에 추정
  • Mesh Reconstruction: 3D 점군을 삼각형 메쉬로 변환하여 매끌러운 표면 생성
이 목록을 처음부터 외울 필요는 없습니다. 학기 중에 하나하나 배워가면서 자연스럽게 액활될 것입니다. 지금은 "아, 이런 것들을 배우게나" 정도로 파악하세요.

응용 분야 (Applications)

컴퓨터 비전 기술이 실제로 활용되는 다양한 분야를 살펅봉니다. 이론이 어디에서 쓰이는지 알면 학습 동기가 높아집니다.
주요 응용 분야

Robotics (로봇)

로봇이 환경을 인식하고 물체를 집는 등의 작업을 수행하려면 컴퓨터 비전이 필수입니다. 예: 공장 자동화 로봇의 부품 인식, 서빙 로봇의 장애물 회피.


Medical Applications (의료 응용)

의료 영상(X-ray, CT, MRI, 병리 슬라이드 등)을 분석하여 질병을 진단하는 데 활용됩니다. 암 세포 검출, 장기 분할 등이 대표적입니다.


Metaverse (메타버스)

사람의 몸을 3D로 재구성하거나, 아바타를 생성하는 기술에 컴퓨터 비전이 활용됩니다.


Driving (자율주행)

카메라로 도로 상황을 파악하고, 보행자/차량/신호등을 분류하고, 장애물과의 거리를 추정합니다. 자율주행의 "눈"이 바로 컴퓨터 비전입니다.


Mobile Devices (모바일 기기)

스마트폰 카메라의 야간 모드, 인물 보정, 파노라마 촬영 등 모두 컴퓨터 비전 기술의 산물입니다.


Accessibility (접근성)

시각 장애인을 위한 이미지 설명, SNS 자동 alt-text 생성 등에 활용됩니다.

컴퓨터 비전은 우리 일상에 이미 깊숭이 스며들어 있습니다. 스마트폰으로 사진 찍을 때마다 컴퓨터 비전 알고리즘이 동작하고 있습니다!

Research.com은 H5-index를 기반으로 컴퓨터 과학 분야의 주요 학회를 랭킹합니다. 상위 10개 학회는:

H5-index란? 최근 5년간 발표된 논문 중, 최소 h편 이상 인용된 논문이 h편 이상인 가장 큰 h 값입니다.

수식으로 표현하면:

\( h\text{-index} = \max_i \min(f(i), i) \)

여기서 \( f(i) \)는 i번째 논문의 인용 수입니다.

CS 전체 학회 중에서 상위 4개 중 3개가 컴퓨터 비전 학회(CVPR, ICCV, ECCV)입니다. 이는 컴퓨터 비전이 현재 컴퓨터 과학에서 가장 활발한 연구 분야 중 하나라는 증거입니다.

저널 랭킹 검색 (Journal Ranking Search)

학회(conference) 외에 저널(journal, 학술지)의 랭킹을 확인하는 방법도 알아둘시다. 연구자로서 논문을 어디에 제출할지 판단할 때 유용합니다.
상세 설명

Journal Citation Reports (JCR)는 Clarivate사에서 운영하는 세계적으로 가장 권위 있는 저널 랭킹 서비스입니다.


주요 기능:

  • Impact Factor: 해당 저널에 발표된 논문이 평균적으로 얼마나 많이 인용되는지를 나타내는 지표
  • Journal 검색: 분야별, 출판사별, 국가별로 저널을 검색 가능
  • Category 랭킹: 특정 분야 내에서 저널의 순위를 확인

링크: https://jcr.clarivate.com/


저널에 논문을 게재하려면 보통 수개월~1년 이상의 심사 기간이 필요합니다. 반면 학회는 발표 후 바로 공개되로, 빠른 연구 공유가 가능합니다.
